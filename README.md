# BookReview App Frontend
This project is a simple book review application frontend built using React. Users can sign up, log in, search for books using the Google Books API, and add reviews for books.

# Getting Started


To get started with the project, clone this repository to your local machine and run the following commands:


**npm install**

**npm start**

This will start the development server and open the application in your default browser at http://localhost:3000.



# Features

**User Authentication:** Users can sign up and log in to the application.

**Book Search:** Users can search for books using the Google Books API.

**Book Reviews: **Users can add reviews for books.


 # Technologies Used

 
**React: **Frontend framework for building user interfaces.

**React Router:** Library for routing in React applications.

**Axios:** Promise-based HTTP client for making API requests.

**Bootstrap:** Frontend framework for building responsive and mobile-first websites.

# Available Scripts


In the project directory, you can run:

**npm start:** Runs the app in development mode.

**npm test:** Launches the test runner in the interactive watch mode.

**npm run build:** Builds the app for production to the build folder.

**npm run eject:** Ejects the project, allowing full control over configuration files (not recommended).


# Deployment
This project is set up for deployment to platforms like Netlify or Vercel. Simply run npm run build and deploy the build folder.
